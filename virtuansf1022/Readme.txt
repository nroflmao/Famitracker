
VirtuaNSF 1.0.2.2 (C)2004- Norix

========
= 概要 =
========

VirtuaNES2(仮称)のサウンドエンジンの検証等の為に作ったNSFプレイヤーです。
多機能なものは全く考えていません。

============
= 動作環境 =
============

OS:	Windows98/2000/XP (基本的にNT系統で使ってください)
	DirectX7.0a以上
CPU:	Pentium200MHz位？速いに越したことはありません。
SOUND:	DirectXに対応したサウンドカード

================
= インストール =
================

アーカイブを適当なフォルダにディレクトリ付きで展開してください。
アンインストールはインストールしたフォルダごと削除すればOKです。

============
= 使用方法 =
============

野生の直感で；

==============
= 予定は未定 =
==============

・ファミコンのバージョンに合わせたバランスを入れてみたい

========
= 謝辞 =
========

・VRC7には[OK]氏のemu2413を利用させて頂きました。
